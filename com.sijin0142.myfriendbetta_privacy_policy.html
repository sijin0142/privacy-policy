제1조 개요

내 친구, 베타❤는 베타 물고기(Betta splendens)를 사랑하는 분들을 위한 개인 기록 관리 앱입니다. 개발자(이하 "운영자")는 사용자의 개인정보를 매우 중요하게 여기며, 「개인정보 보호법」 및 관계 법령을 준수합니다.

본 방침은 앱 사용 중 수집되는 정보의 항목·이용 목적·보관 기간·제3자 제공 여부· 이용자 권리 등을 안내합니다.

✅ 본 앱은 계정 가입·로그인 기능이 없으며, 사용자가 입력한 데이터는 기기 내부에만 저장됩니다. 외부 서버로 개인 데이터가 전송되지 않습니다.

제4조 개인정보의 제3자 제공

운영자는 아래의 경우를 제외하고 사용자의 정보를 제3자에게 제공하지 않습니다.

Google AdMob (광고 서비스)

  • 제공 정보: 광고 식별자, 기기 정보, 앱 사용 정보
  • 제공 목적: 광고 게재 및 성과 분석
  • 보유 기간: Google의 데이터 보유 정책에 따름
  • 관련 정책: Google 개인정보처리방침 / AdMob 정책
ℹ️ 현재 앱은 테스트 광고 ID를 사용 중입니다. 정식 출시 전 실제 AdMob 광고 단위 ID로 교체 예정입니다.

데이터 종류보유 기간삭제 방법
베타 정보 및 기록 (DB) 사용자가 삭제하거나 앱 삭제 시까지 앱 내 삭제 기능 또는 앱 제거
베타·수조 사진 (filesDir) 베타/수조/사진 삭제 시 또는 앱 삭제 시 앱 내 삭제 기능 또는 앱 제거
앱 내 설정 (SharedPreferences) 앱 삭제 또는 데이터 초기화 시까지 Android 앱 데이터 삭제

제6조 만 14세 미만 아동의 개인정보 보호

본 앱은 만 14세 미만 아동을 대상으로 개인정보를 의도적으로 수집하지 않습니다. 만 14세 미만 아동이 앱을 이용하는 경우, 법정대리인의 동의가 필요합니다. 운영자가 만 14세 미만 아동의 정보를 수집한 사실을 알게 된 경우 즉시 삭제합니다.

제7조 이용자의 권리

사용자는 다음의 권리를 행사할 수 있습니다.

  • 열람권: 앱 내에 저장된 모든 데이터를 앱 화면을 통해 직접 확인 가능
  • 수정권: 베타 정보 수정 화면을 통해 언제든 수정 가능
  • 삭제권: 베타 삭제·용궁 앨범 삭제·사진 삭제 기능 제공
  • 처리 정지권: 앱 삭제 시 기기 내 모든 데이터 완전 삭제
  • 광고 개인화 거부: Android 설정 > Google > 광고 > 광고 ID 재설정 또는 광고 개인화 사용 중지

제8조 개인정보 보호 기술적·관리적 조치

  • 모든 데이터는 기기 내부 저장소(filesDir, Room DB)에 암호화 없이 저장되며, 타 앱에서 접근할 수 없는 앱 전용 영역을 사용합니다.
  • 외부 서버 전송 없이 100% 로컬 저장으로 네트워크 해킹 위험에서 자유롭습니다.
  • 이미지 파일은 압축 처리(JPEG 80% 품질)하여 저장합니다.
  • 앱 삭제 시 앱 전용 저장소 내 모든 데이터가 Android OS에 의해 자동 삭제됩니다.

제9조 광고 서비스 안내

본 앱은 무료 서비스 운영을 위해 Google AdMob을 통한 광고를 제공합니다.

  • 배너 광고: 각 화면 상단에 표시 (크기: 320×50 Banner)
  • 전면 광고: 베타 등록 5회마다 1회 표시 (InterstitialAd)
  • 광고를 통한 수익은 앱 유지 및 개선에 사용됩니다.
  • 광고 개인화를 원하지 않으시면 Android 설정 또는 Google 광고 설정에서 조정할 수 있습니다.

제10조 알림 (푸시) 서비스

본 앱은 물갈이 기록 후 7일 경과 시 로컬 알림(WorkManager)을 통해 물갈이 주기를 안내합니다. 이 알림은 외부 서버와 무관하며 기기 자체에서 발송됩니다.

알림 수신을 원하지 않으시면 Android 설정 > 앱 > 내 친구, 베타❤ > 알림에서 비활성화할 수 있습니다.

제11조 개인정보처리방침의 변경
